Variáveis e Tipos Primitivos

Idenficadores de variáveis podem começar com letra, $ ou _.

Não podem começar com números.

É possível usar letras ou números.

É possível usar acentos ou símbolos.

Não podem conter espaços.

Não podem usar palavras reservadas: function, var...

Observações

Maiúsculas e minúsculas fazem diferença.

Tente usar nomes que tenham relação com o conteúdo das variáveis.

Tipos de dados

Tipos primitivos primordiais

    • Nan
  • String
  • Boolean

  • undefined
