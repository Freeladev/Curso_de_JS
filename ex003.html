Conversão de tipos de dados

Em Js é possível converter numbers em strings e vice-e-versa. Para o primeiro caso, é necessário usar Number(), na qual os parênteses abarcarão todo o conteúdo à frente. Para o segundo caso, usa-se String() ou toString().

No exemplo abaixo JS vai considerar os dados como strings, msm se tratando de numbers

            
                var n1 = window.prompt('Digite um número:')
                var n2 = window.prompt('Digite outro número:')
                var s = n1 + n2
                window.alert('a soma dos valores é: ' + s)

Já no exemplo abaixo JS vai tratar os dados como numbers:

            
                var n1 = Number(window.prompt('Digite um número:'))
                var n2 = Number(window.prompt('Digite outro número:'))
                var s = n1 + n2
                window.alert('a soma dos valores é: ' + s)
